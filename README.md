# IT-Web Tools

Eine Sammlung einfacher IT-Tools die direkt im Browser laufen – kein Login, keine Installation.

## Tools

- **Subnetzrechner** – IP-Adresse und CIDR eingeben, Netzwerkadresse, Subnetzmaske, Broadcast und nutzbare Hosts werden berechnet
- **Zahlensystemrechner** – Zahlen zwischen Binär, Dezimal, Oktal und Hexadezimal umrechnen
- **Passwort Generator** – Zufällige Passwörter mit wählbarer Länge generieren

## Mitmachen

Jeder darf gerne mitmachen! Einfach das Repo forken, Änderungen machen und einen Pull Request stellen.
